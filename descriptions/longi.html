Project Description

For my masters thesis I will be analyzing publicly available data from the CDC COVID Data Tracker and GISAID Data. Utilizing this data, I aim to assess whether SARS-CoV-2 variant proportion data from state level surveillance can detect changes in COVID-19 case rates during a period of rising Delta variant prevalence (June 2021-December 2021), while accounting for state vaccination rates, seroprevalence, and age distribution. If effective, this could potentially be used as an indicator for surges in COVID-19 cases.

This is an ongoing project but the linked github repository has the most up to date information.
